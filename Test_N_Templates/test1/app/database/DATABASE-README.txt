This is the folder for our practice/template database.

This practice database is a student database with homework requests.

Structure is as follows:
    Student --> Class (Class List) --> Homework (Homework List, Boolean Completion)
    
    Example:
    Payton -->  Math      -->   Worksheet 1, True (Done)
                                Worksheet 2, False (Not Done)
                English   -->   Essay, False
                Physics   -->   Capacitors 1, True
                                Kirchoff 1, True
